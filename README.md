# JavaScript - Exercícios e Aulas Práticas

Repositório criado para praticar e consolidar os conceitos fundamentais de JavaScript. Contém diversos exercícios e exemplos práticos estudados durante aulas e treinamentos.

## Tecnologias
- JavaScript
- HTML
- CSS (quando aplicável)

## Objetivo
Praticar lógica de programação, manipulação de arrays, funções, loops e outros conceitos básicos de JavaScript. Ideal para reforço de aprendizado e experimentação de código.

## Estrutura do Repositório
- Exercícios separados por tema (arrays, funções, loops, etc.)
- Pequenos exemplos e testes práticos
- Anotações e soluções comentadas

> Este repositório é voltado para **estudo e prática**, e serve como registro da evolução no aprendizado de front-end.
